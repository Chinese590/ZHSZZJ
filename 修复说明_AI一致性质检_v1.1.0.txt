数据堂质检工具 AI一致性质检助手 v1.1.0
==========================================

本版本直接在原质检工具中增加 AI 一致性质检，不另建程序。

新增功能：
1. 原图与结果图本地一致性初筛。
2. 主体一致性、结构完整性、细节保留、颜色一致性、文字与Logo、
   背景符合性、画面质量、版面协调性八项检测。
3. 一致性评分、低/中/高风险和建议通过/复核/返修。
4. OpenAI、Gemini、OpenAI兼容自定义API在线深度复核。
5. 本地中高风险智能触发在线复核，同时保留手动复核按钮。
6. AI问题标签和返修备注可自动推荐，也可人工采纳、修改。
7. AI检测结果审计日志和Excel“AI辅助统计”工作表。
8. 运行AI时禁止移动、删除或提交当前数据组，避免线程读取冲突。

重要约束：
- AI不会自动判定通过或不通过。
- AI不会自动删除或移动文件夹。
- AI不会静默覆盖人工已填写的返修备注。
- 在线复核会发送两张图片与指令到所选服务商。
- API Key不写入项目日志或Excel。

GitHub使用方法：
1. 解压本更新包。
2. 将包内文件和文件夹复制到GitHub Desktop本地仓库根目录。
3. 出现提示时选择替换目标文件。
4. Summary填写：增加AI一致性质检助手。
5. Commit to main并Push origin。
6. GitHub Actions运行 Release Stable DataTang QC Tool。
7. 版本填写：1.1.0。
8. v1.0.9新版启动器会提示更新，点击立即更新即可；运行库依赖没有变化时
   只下载app.zip，不重复下载Python运行库。
