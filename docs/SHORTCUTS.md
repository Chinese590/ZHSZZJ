# 快捷键操作说明

数据堂质检工具支持左手单键、数字键盘和自定义快捷键。按 `F1` 或点击顶部“快捷键设置”可查看并修改，保存后立即生效。

## 默认快捷键

| 功能 | 左手/主快捷键 | 数字键盘/备用快捷键 |
|---|---|---|
| 上一组数据 | `A` | `Num+4` |
| 下一组数据 | `D` | `Num+6` |
| 质检通过并进入下一组 | `Space` | `Num+5` |
| 进入返修备注 | `X` | `Num+0` |
| 提交返修并进入下一组 | `Enter` | `Num+Enter` |
| 聚焦返修备注 | `R` | — |
| 本地 AI 检测 | `Q` | `Num+1` |
| 在线深度复核 | `E` | `Num+3` |
| 采纳 AI 标签 | `1` | — |
| 采纳 AI 备注 | `2` | — |
| 清空当前 AI 结果 | `3` | — |
| 两张图片适应窗口 | `F` | — |
| 当前图片切换为 1:1 | `V` | — |
| 切换原图/结果图焦点 | `Tab` | — |
| 放大当前图片 | `+` / `=` | — |
| 缩小当前图片 | `-` | — |
| 撤销上一步 | `Z` | — |
| 刷新队列 | `C` | — |
| 退出输入状态 | `Esc` | — |
| 查看/设置快捷键 | `F1` | — |
| 打开 AI 设置 | `F2` | — |
| 删除当前数据组 | `Ctrl+Delete` | — |

## 快速质检流程

通过图片：

```text
Space → 通过并自动显示下一组
```

返修图片：

```text
X → 输入返修备注 → Enter → 提交返修并显示下一组
```

返修备注需要换行时使用 `Shift+Enter`。

## 防误触规则

- 中文指令、英文指令及普通输入框获得焦点时，字母、数字、空格和方向操作不会触发快捷键。
- 返修备注框中只有“提交返修”快捷键保持启用；`Shift+Enter` 仍用于换行。
- 删除数据组继续执行原有两次确认，不会因为按下快捷键直接删除。
- AI 检测进行中时，原有文件流转按钮和对应快捷键仍受业务状态限制。
- 相同按键不能同时绑定到多个功能；设置窗口会阻止冲突配置保存。
