# 分阶段执行状态

## 第一阶段：启动器与在线运行库

- 缓存地址选择、持久化、断点下载、SHA256 和原子安装完成。
- 新版本提示、一键更新、按组件更新和离线回退完成。
- Windows Actions 编译和发布工作流完成。

## 第二阶段：完整质检业务

- 目录扫描、图片与色彩显示、指令编辑、文件流转、回收站删除、撤销、日志、日报和实时刷新完成。
- 质检文件操作仍由人工最终确认。

## 第三阶段：AI 一致性质检助手

- 本地主体区域、结构、颜色、细节、画质和版面初筛完成。
- 可选 DINOv2 ONNX 主体语义相似度完成；模型下载失败不影响基础质检。
- OpenAI、Gemini、OpenAI 兼容自定义 API 深度复核完成。
- 本地中高风险智能触发及手动在线复核完成。
- AI 问题标签、返修备注、建议结论、审计日志和 Excel 统计完成。
- AI 不执行通过、不通过、删除或移动文件夹。
